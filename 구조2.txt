아래 A,B,C 해당 기능들은 주석처리하거나 숨겨줘.

A 뱃지 기능
B 캐릭터(참가자) 프로필 '나이'
C 캐릭터 비밀번호
----------------

추가 기능
- 게임 승리한사람 고를 때, 복수로 선택할 수 있게 변경
- 홈에서 게임 눌렀을 때, 배경음악칸에 파일명 표시하고, 재생은 없애줘, 게임 시작을 눌렀을 때, 해당 화면에서 자동으로 배경이 루프로 재생되면 됨.
참가자 등록 란 하단 참가자 생성 버튼 오른쪽에 게스트 생성 버튼 추가 (게스트는 1회용, 생성시 이름만 입력)
- 게임 시작 전, 배경음악 윗칸에 배경이미지 도 파일명 표시해주고, 배경이미지 선택(교체)할 수 있게도 해줘.
- 해당 버튼들의 이름(텍스트를) 코드의 어느 부분에서 바꿀 수 있는 지 알려줘. 추가로 버튼 색상코드도 조절할 수 있게 알려줘.
- 게임시작 후, 설정한 배경화면이 해당 브라우저의 높이나 위치에 따라 사이즈가 알맞게 조정되게 해줘. 상하좌우로 블랙바가 생겨도 괜찮아.
- 코드 내에서 캐릭터의 정보를 추가하거나 삭제할 수 있는 곳이 어디인지 알려줘 직접 하게.

- 게임 화면 내 참가자들의 이름은 우측 상단이 아니라, 화면 최하단 중앙에서 두번째 칸부터 중앙정렬로 있으면 좋겠어.
- 그리고 게임 종료는 그 아래 중앙정렬로 있게 해줘, 게임 종료 왼쪽에 시계 아이콘 버튼 생성해주고, 누르면 팝업 형태로 스톱워치랑 타이머 중에 선택할 수 있게 해주고, 타이머 누르면 3,5,10,15분 세팅값 설정하거나 직접 숫자 입력해서 세팅할 수 있게.
그리고 스탑워치 누르면 바로 0초부터 시작할 수 있게. 물론 정지랑 종료는 둘다 있어야 함. 타이머 끝나면 미리 세팅된 기본 음성이 나옴. 해당 음성 경로도 알려줘.
- 게임 내에 타이머 넣을거니까, 메인탭에 타이머는 없애줘.
- 그러면 홈탭에는 게임, 캐릭터, 관리자가 남고, 최초에 관리자를 킬 때 비밀번호를 입력할 수 있게 해줘. 그 다음부터 관리자를 킬 때는 비밀번호를 입력하고 사용할 수 있게 해.
- 게임 정보에 허용 인원 (최대)은 없애주고 그냥 '비고(인원 등)' 칸(텍스트 기입)으로 바꿔줘, 알아서 세팅할 수 있게.
- 새 캐릭터 등록할 때 이미지 등록도 만들어줘. (가져오기 등) 그리고 기본 이미지도 한 9개정도 만들어줘. 

- 기본 캐릭터 이미지 9개와, 뒷배경 색상 9가지 정도 넣을거야. 배경 색상 선택은 가장 기본적인 색중에 톤다운된 컬러로 알아서 만들어주고, 기본 캐릭터 이미지 9개는 자동으로 만들어주되(가능하다면), 나중에 폴더에서 교체할 수 있게 해줘.(캐릭터 이미지 디자인은 보드게임 디자이너 Weberson Santiago의 화풍(보드게임 쿠, 아발론빅박스 등)을 참고해줘,.
캐릭터 생성 시, 이미지를 선택하지 않으면 자동으로 생성해줘.
기본 캐릭터 이미지의 기본 9가지 컨셉은 아래와 같다.
- 왕
- 기사
- 여관주인
- 휴머노이드
- 마법사
- 음유시인
- 역병의사
- 노숙자(거지)
- 귀족
