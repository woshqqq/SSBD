첫화면 게임 선택에서 없는 게임도 생성할 수 있게 버튼 만들어놓을 것
- 게임 정보 추가 (장르/허용 인원 숫자/) 게임 장르는 미리 세팅되어 있어야 함. (코드 or env 파일)
- 생성할때도 장르 선택하고, 허용 인원 인수 설정.

관리자(수정)모드도 만들 것
관리자 모드 구성
게임 수정/생성하기 (게임 이름 및 정보 변경/ 배경음악 연결하기 버튼)

그렇게 게임을 선택해서 들어가면,

- 참가자 등록 ( 프리셋 불러오기 & 참가자 생성 버튼 ) 버튼
- 게임 시작 버튼
- 리소스 관리 버튼(서포트 도구)
	리소스관리버튼에서는 리소스 개수(기본5개)를 변경할 수 있음. 0개부터 10개 사이(리소스는 이후 게임 시작 시 하단에 보임. 리소스 미선택시(0개) 미노출)
- 참가자 등록을 1명 이상 하지 않으면 게임 시작을 누르지 못함.
- 사용자가 참가자 등록을 한 다음 게임 시작을 누르면 게임이 시작되고, 사전에 설정한 배경이미지가 있다면 해당 이미지를 띄워주고, 없다면 무조건 background_default.png를 띄움.(사전에 미리 넣어둘 것임)
- 

게임 시작시 노출되는 화면 구성 안내
- 맨 뒤 배경
- 맨 위에 작게 1초부터 타이머 시작 00:00
- 음소거 버튼은 작게 상단 타이머 옆에 불투명도 30, 
- 해당 참여자들의 이름을 우측 상단에 배치, 알파벳, 가나다 순서대로 나열. 참여자 누르면 팝업으로, 수정 / 승리 버튼이 팝업 표시.
수정 누르면 다른 프리셋(참여자)으로 교체할 수 있음. 승리버튼을 누르면 해당 참석자의 승리 횟수 1회 추가, 그리고 이펙트 효과 발생 추후에 2d 이미지나 3d 이미지 삽입하여, 해당 애니메이션에 해당 캐릭터의 이미지를 삽입. 
예시 ) 커튼이 겆히며 줌인 애니메이션, 그리고 해당 캐릭터의 초상화가 액자에 걸려있고, 상단에 '승리! 명예의전당 등록!' (텍스트는 나중에 코드상에서 수정할 수 있게 함) (여기서 필요한 자료(애니메이션 등)가 있다면  알려줘)
- 게임 종료 버튼 (하단 좌측)
	버튼 클릭시 승리자 체크를 하시겠습니까? 예/아니오
	예 클릭시 해당 게임 참여자 목록 팝업 노출, 아니오 클릭시 그대로 게임/종료
게임종료시 해당 게임 결과 노출 
게임 이름/ 게임 정보(간략) / 참석자 목록 / 승리횟수 / 게임플레이 시간 / 우승자(승리횟수 가장 높은 사람)
하단에 버튼 '게임 선택창으로' 누르면 게임 선택(홈)으로 이동


-캐릭터 정보
캐릭터 이미지 기입가능하게 설정. 해당 이미지는
1. 뱃지는 이미지파일로 삽입할 예정.
뱃지는 미리 만들어두고, 특정 요건을 충족하면 부여.

예시 ) 마피아게임 10회 우승 -> 대부 뱃지 수여
그냥 10회 우승 -> 즐거운 게이머 뱃지
100회 참여 -> 현생은 괜찮으신가요? 뱃지


트로피는 삭제. 참여횟수는 게임 시작시 참여한 인원수에게 자동으로 +1씩 

캐릭터 창에 수정 삭제 버튼 추가,
삭제할때는 팝업띄우기(삭제하시겠습니까?)
수정할때는 최초 or 사전에 입력한 값대로 비밀번호 입력해야 수정 가능.
수정에는 캐릭터 정보 및 승리, 참여, 횟수  /뱃지 추가 및 삭제 수정
